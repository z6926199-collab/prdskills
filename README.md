# PRD Skills v3.1

> SkillOpt 3 轮 Epoch 训练优化 + 飞书产品规划部模板对齐 + 统一理解确认流程

**触发方式**：用户输入 `/tcl-cpghb-prd-v3` 时激活。

**密码**：请联系作者获取压缩包密码。

---

## 核心能力

- **全量撰写**：从需求背景到完整 PRD 一站式生成
- **早期探索**：帮助梳理模糊想法，输出概念版 PRD 并冻结范围
- **局部修改**：对已有 PRD 做精确修改，不破坏原有结构
- **PRD 评审**：8 大类 Review + 5 大类风险自审
- **证据追溯**：为原始素材分配编号，在 PRD 中标注来源
- **多产品线支持**：通过域文件注入不同产品线的上下文和规范
- **个性化学习**：从用户历史文档中提取写作风格，持续贴合团队习惯

## v3.1 新增

- **统一理解确认**：所有输入先理解再追问，追问聚焦功能歧义而非元信息
- **埋点统一到 6.1**：5.3 具体表现不含埋点，减少单元格内容量
- **飞书模板对齐**：6.x 子节顺序与部门模板一致
- **场景树深度强化**：正常路径+异常分支(≥2)+终态，含 5.3 映射规则
- **产出门控 6a**：6 项自评（路由/子块/边界/一致性/无臆造/去AI味）
- **标题无括号注释**：PRD 输出标题干净
