# PRD Markdown 模板

本文件是完整 PRD 结构的唯一权威，严格映射飞书《XX需求文档【产品规划部模板】》。

源模板：
`https://sgpr1076gv.feishu.cn/docx/KULldU2p3oz8HjxiHh6csA4Znxf`

## 使用规则

1. 保留全部章节、字段名、顺序和表格列数。
2. 不新增 `5.2.1`、证据追溯表、评审更新记录等模板外章节。
3. 缺失信息填 `待确认`；确认不涉及时填 `不涉及（原因）`。
4. 用户要求精简时可以压缩内容，但不得删除标记为“必备”的章节。
5. Markdown 不支持飞书的合并单元格。5.3 用“字段 / 功能描述 / 原型图”三列表头做等价映射。
6. 每个独立功能单元重复一张 5.3 三列九字段表，不得增加或删除字段行。
7. 补充材料放在完整 PRD 之后并标记“模板外补充材料”，不得混入模板正文。
8. 交付前检查：一级章节依次为一至七和附录；不出现 5.2.1、6.11 等新增编号；6.7 包含玩机技巧、NXTPAPER、应用分身、去TCL化。
